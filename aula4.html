Aula 4 - Criando os demais arquivos html

Abaixo iremos criar os arquivos index.html, contato.html e sobre.html. Arquivos que usarão a base.html para funcionar

Abaixo o index.html

Explicações:
  • {% extends "base.html" %} - aplica como base o arquivo indicado
  • {% block title %}Página inicial{% endblock %} - troca o título em base.html
  • {% block content %} - indica o conteúdo que será carregado em base.html
  • {% endblock %} - encerra as alterações aplicadas pelo Flask (Python)
