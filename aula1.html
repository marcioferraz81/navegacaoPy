Aula 1 - Instalação necessária

Python

https://www.python.org/downloads/

Para desenvolver com Django, necessitamos da linguagem Python instalada. Use o link acima para fazer download, execute o instalador e ao fazer a instalação marque a opção "Add python.exe to PATH".

Após a instalação, para ver se foi instalado corretamente abra o prompt de comando e digite: python --version

Resultado da execução do comando.

Essa tela indica que o python está funcionando corretamente.

Para testar se o gerenciador de pacotes pip esta funcionando corretamente, digite no prompt de comando: pip --version

Resultado da execução do comando.
