Navegação - Python (Flask)

O objetivo deste curso é iniciar o desenvolvedor na criação de aplicações Web usando o Framework Flask (Python).
Este curso foi criado pelo Professor Marcio Rogerio Santos Ferraz (ETEC Martinho Di Ciero - Itu), e disponibilizado na internet em Setembro de 2024.

Em caso de dúvidas sobre o curso, entre em contato com o criador do curso pelo e-mail: marcio.ferraz@etec.sp.gov.br

Requisitos e instalação de softwares (veja aula 1):

- Python (https://www.python.org/downloads/)

Pré-requisitos:

Para que o participante tenha um melhor aproveitamento é importante que já tenha conhecimentos de lógica de programação, HTML , CSS e Python.

Público Alvo:

professores e funcionários do CPS.

Objetivo:

Dar os primeiros passos no desenvolvimento de aplicações Web usando Framework Flask (Python).

Duração: 20 horas.

Certificação: o certificado só será emitido após o participante ter enviado o trabalho solicitado no item atividade, para a CETEC, aos cuidados do Prof. Oswaldo e enviar também o questionário informativo, devidamente preenchido, recebido no momento da inscrição, para o seguinte e-mail: ogiorgi@hotmail.com .

IMPORTANTE:

Para que sua tarefa seja avaliada e certificada é necessário que você execute todos os passos como propostos no curso, e a seguir envie o seu trabalho concluído para a CETEC aos cuidados do Prof. Oswaldo. 

Observação:

O curso inicia-se quando você recebe o e-mail com o endereço do provedor que hospeda o curso e não tem prazo para ser concluído. O que determina a data de encerramento é o seu tempo disponível, ou seja, a sua inscrição não caduca. Para a CETEC o importante é que você conclua o curso.
